## Project Overview
This project demonstrates a cloud-based database application built with MySQL and Flask. The application allows users to manage tasks through individual lists, with backend logic handling database operations.

## My Contributions
- Designed and created the database schema, including tables and columns.  
- Connected the MySQL database to the backend using Flask.  
- Implemented backend logic for SQL operations (INSERT, DELETE, UPDATE, SELECT) through Flask.  
- Developed functionality for managing individual tasks within specific lists, enabling task creation, updates, and deletions.

## Software Description
This is an web-based to do list application that communicates with a SQL server (preferably PostgreSQL or MySQL). The goal is to design a user interface that enables a visitor to register an account, log in to their account, create multiple different to-do lists, and modify each list. Each one of these requests will be funneled back to an API server that acts as the man in the middle between our server/webserver and our database.
